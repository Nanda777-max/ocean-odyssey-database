## 📖 Overview

Ocean Odyssey is a fictional cruise management company used to demonstrate the design and implementation of a complete database system.

This project focuses on building a robust relational database capable of managing cruise operations, passenger information, bookings, ships, cabins, and related business processes. It also includes a MongoDB implementation to demonstrate document-oriented data modelling.

The project showcases the complete database development lifecycle—from schema design and data population to SQL querying and NoSQL integration.

## 🚀 Getting Started

### Prerequisites

* Oracle Database
* Oracle SQL Developer
* MongoDB Community Edition

### Running the Project

1. Clone the repository.
2. Execute the SQL schema creation script.
3. Populate the database using the sample data scripts.
4. Run the SQL query scripts.
5. Execute the MongoDB script to create and query the document collection.
